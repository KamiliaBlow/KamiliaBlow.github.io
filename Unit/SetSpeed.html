Unit Methods

Method Unit:SetSpeed [-] [+]

Sets the Unit's speed of given [UnitMoveType] to given rate. If forced, packets sent to clients forcing the visual change.

enum UnitMoveType
{
    MOVE_WALK           = 0,
    MOVE_RUN            = 1,
    MOVE_RUN_BACK       = 2,
    MOVE_SWIM           = 3,
    MOVE_SWIM_BACK      = 4,
    MOVE_TURN_RATE      = 5,
    MOVE_FLIGHT         = 6,
    MOVE_FLIGHT_BACK    = 7,
    MOVE_PITCH_RATE     = 8
};

Синапсис

Unit:SetSpeed( type, rate )

Unit:SetSpeed( type, rate, forced )

Аргументы

UnitMoveType type
See method description.
number rate

Valid numbers: all decimal numbers.

boolean forced (false)
See method description.

Возвращает

Nothing.
