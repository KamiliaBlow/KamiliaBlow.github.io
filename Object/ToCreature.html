Object Methods

Method Object:ToCreature [-] [+]

Attempts to convert the Object to a Creature.

If the Object is not a Creature, returns nil.

Синапсис

creature = Object:ToCreature()

Аргументы

Нету.

Возвращает

Creature creature

The Object as a Creature, or nil.
