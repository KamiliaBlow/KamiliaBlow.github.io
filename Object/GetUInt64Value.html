Object Methods

Method Object:GetUInt64Value [-] [+]

Returns the data at the specified index, casted to an unsigned 64-bit integer.

Синапсис

value = Object:GetUInt64Value( index )

Аргументы

number index

Valid numbers: integers from 0 to 65,535.

Возвращает

number value

Valid numbers: integers from 0 to 18,446,744,073,709,551,615.
