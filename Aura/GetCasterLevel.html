Методы Aura

Метод Aura:GetCasterLevel [-] [+]

Возвращает уровень Юнита который произнес Заклинания вызвавшее применение этой Ауры.

Синапсис

caster_level = Aura:GetCasterLevel()

Аргументы

Нету.

Возвращает

number caster_level

Действительные числа: целые числа от 0 до 4,294,967,295.

S

Поисковые приемы

Префикс поиска с типом, за которым следует двоеточие (напр. fn:) чтобы ограничить поиск заданным типом.

Принимаются следующие типы: fn, mod, struct (или str), enum, trait, typedef (или tdef).
