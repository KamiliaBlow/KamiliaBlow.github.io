Eluna Lua Engine© API

API Eluna Lua Engine© позволяет добавлять свой собственный Lua-код, который будет выполняться при срабатывании определенных событий (так называемых "хуков").

Добавьте новую игровую команду, дайте жизнь мобам с новым кастомным поведением или например подожгите игроков, пытающихся устроить дуэль! Если хук существует, вы можете написать для него сценарий.

О Eluna

Eluna это Lua интерпретатор для эмулятора World of Warcraft. Eluna поддерживает TrinityCore (только мой репозиторий).

Чтобы получить Eluna, просто клонируйте репозиторий из МОЕГО аккаунта GitHub. В форке этого репозитория Eluna уже интегрирована, так что вам нужно просто скомпилировать ее и начать кодить!

Классы

Aura

Эффект Заклинания который остается на Юните после того как Заклинание было использовано.
BattleGround

Содержит состояния для полей сражений.
Corpse

Состояние, когда Игрок умер.
Creature Юниты не контролируемые Игроком (т.е. NPC).
ElunaQuery

Результат запроса к базе данных.
GameObject

Наследует все методы от: Объект, Мировой Объект
Global

Эти функции можно использовать в любом месте и в любое время, в том числе при запуске.
Group

Наследует все методы от: нету
Guild

Наследует все методы от: нету
Item

Наследует все методы от: Объект
Map

Игровая карта, например, Азерот, Восточные королевства и т.д.
Object

Базовый игровой объект (либо Предмет, либо Мировой Объект).
Player

Наследует все методы от: Объект, Мировой Объект, Юнит
Quest

Наследует все методы от: нету
Spell

Экземпляр заклинания, создаваемый, когда заклинание произносится Юнитом.
Unit

Наследует все методы от: Объект, Мировой Объект
Vehicle

Наследует все методы от: нету
WorldObject

Наследует все методы от: Объект
WorldPacket

Пакет, используемый для передачи сообщений между сервером и клиентом.

Поисковые приемы

Префикс поиска с типом, за которым следует двоеточие (напр. fn:) чтобы ограничить поиск заданным типом.

Принимаются следующие типы: fn, mod, struct (или str), enum, trait, typedef (или tdef).
