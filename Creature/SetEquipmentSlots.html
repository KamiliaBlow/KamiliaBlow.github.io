Creature Methods

Method Creature:SetEquipmentSlots [-] [+]

Equips given Items to the Unit. Using 0 removes the equipped Item

Синапсис

Creature:SetEquipmentSlots( main_hand, off_hand, ranged )

Аргументы

number main_hand

Main hand Item's entry.

Valid numbers: integers from 0 to 4,294,967,295.

number off_hand

Off hand Item's entry.

Valid numbers: integers from 0 to 4,294,967,295.

number ranged

Ranged Item's entry.

Valid numbers: integers from 0 to 4,294,967,295.

Возвращает

Nothing.
