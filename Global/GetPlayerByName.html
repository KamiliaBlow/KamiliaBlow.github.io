Global Methods

Method Global:GetPlayerByName [-] [+]

Finds and Returns Player by name if found

In multistate, this method is only available in the WORLD state

Синапсис

player = GetPlayerByName( name )

Аргументы

string name

Name of the Player.

Возвращает

Player player
See method description.
